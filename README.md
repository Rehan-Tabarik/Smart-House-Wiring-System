# Smart House Wiring System

## Overview

This project presents the design and simulation of a residential electrical system using MATLAB and Simulink. The project focuses on load distribution, system planning, and structural modeling for efficient electrical implementation.

## House Design Model

The house design model represents the structural layout of the residential building. It serves as the foundation for planning the electrical system by defining room arrangement, space utilization, and the overall framework for load distribution.
